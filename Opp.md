Below are Markdown tables grouped by **category**. Each table lists the technology, where in the audit life‑cycle it adds value, and a concise idea for introducing or expanding AI.

## 1. Core Systems (High‑volume daily use & critical path)

| Technology                            | Audit Phase        | AI Opportunity                                                                                   | Short Description                                                    |
| ------------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------- |

| **Levvia**                            | Fieldwork          | Predictive planning assistant for smaller engagements based on historical risk patterns.         | Cloud platform for very small audits (VSA).                          |
| **EMS Client**                        | Fieldwork          | Auto‑summarise auditor comments and cross‑link to supporting evidence.                           | Core engagement documentation system (EOL May 2027).                 |

| **Quality Assessment Platform (QAP)** | Quality Mgmt       | AI‑driven gap analysis against ISQM 1 controls.                                                  | ServiceNow GRC solution ensuring ISQM 1 compliance.                  |
| **InsightBox**                        | Analytics Assist   | Scenario‑based GenAI to recommend analytical packages and interpret outputs.                     | Deloitte AU Audit Analytics Platform.                                |
